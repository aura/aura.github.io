Package: Aura\Framework

Using PHPUnit, creates a test file from an existing package source class and places it in the package tests directory.

Usage is ...

 $ php package/Aura.Framework/cli/make-test {$FILE}

... where $FILE is a package file path, e.g. package/Vendor.Package/src/Vendor/Package/Class.php.
